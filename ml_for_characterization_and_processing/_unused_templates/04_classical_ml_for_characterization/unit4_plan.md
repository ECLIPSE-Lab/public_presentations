# Unit 4 Plan — Classical ML for Characterization Tasks (ML-CP)

## Unit intent
Teach students to build robust, interpretable classical ML baselines for characterization and processing problems before deep learning.

## Learning objectives
1. Frame characterization tasks into suitable ML problem types.
2. Build feature-engineering + baseline model pipelines correctly.
3. Select metrics/splits/calibration strategies aligned with lab constraints.
4. Perform structured error analysis and detect leakage/domain shift.

## Dependency links
- Backward: Unit 2 (physics of data), Unit 3 (data readiness)
- Forward: Unit 5 (deep learning methods for the same task family)

## Slide architecture (50)
- 1–12: Task framing and pipeline prerequisites
- 13–24: Core classical model families + evaluation choices
- 25–36: Calibration, interpretation, robustness, reproducibility
- 37–42: Case sketches + comparison with deep methods
- 43–50: Exercise scaffolds + summary + references

## Exercise scaffold
- E1: Define task, split, and baseline metric set
- E2: Benchmark 3 classical models
- E3: Tune + calibrate one model and produce failure analysis
